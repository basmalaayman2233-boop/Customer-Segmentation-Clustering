# Customer Segmentation Using Clustering Algorithms

This project applies unsupervised machine learning techniques to segment customers based on their behavior and spending patterns.

## Dataset
- Customer data containing:
  - Annual Income
  - Spending Score

## Clustering Algorithms Used
- K-Means Clustering
- DBSCAN Clustering

## Techniques Applied
- Elbow Method to determine optimal number of clusters
- Visualization of clusters and centroids
- Density-based clustering using DBSCAN
- Performance evaluation using Silhouette Score

## Evaluation Metrics
- Silhouette Score

## Visualizations
- Elbow Method graph
- K-Means cluster visualization
- DBSCAN cluster visualization

## Technologies
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
